To define a node in LDAP, 
we use keywords such as 
o (Organization), 
ou (Organizational Unit), 
cn (Common Name), 
sn (Surname), 
uid (User Id), 
dn (Distinguished Name), 
dc (Domain Component) etc. 
A user in LDAP can be defined as below.
dn: uid=surya,ou=people,dc=concretepage,dc=com
